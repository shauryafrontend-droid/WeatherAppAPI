# 🌦️ Weather App

This project is a simple Weather Application built using **HTML, CSS, and JavaScript**.
The app fetches real-time weather data from a weather API and displays information like temperature and weather conditions for any searched city.

---

## 🌐 Live Demo

https://weatherdashboardapiproject.netlify.app/

---

## 📌 Features

* Search weather by city name
* Displays real-time weather information
* Shows temperature, humidity, and weather conditions
* Clean and responsive user interface
* Fast API-based weather updates

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Weather API

## ⚙️ How It Works

1. User enters a city name.
2. JavaScript sends a request to the weather API.
3. The API returns weather data in JSON format.
4. The data is displayed on the webpage.

## 🚀 Installation

1. Clone the repository

git clone https://github.com/shauryafrontend-droid/weather-app.git
